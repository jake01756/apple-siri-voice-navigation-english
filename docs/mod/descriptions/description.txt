[orange]Experience the road with Siri![normal] Siri Voice Navigation brings the iconic voices from Apple OS to truck simulator games. With [orange]over 20 voices in English[normal] to choose from, Siri is here to guide your trucking adventures, making your journey more enjoyable and memorable.

This mod is forked and licensed under the GPL-v3.

[orange]Introduction[normal]
I speak English, so having a mod that has a whole bunch of voices instead of just english ones takes up extra space on my system. So, I decided to fork the repo and remove all the voices, except the english ones. If you want to contribute to this project, I suggest you do so upstream.

The English only version is hosted on GitHub at [orange]https://github.com/jake01756/apple-siri-voice-navigation-english[normal].

The main project is hosted on GitHub at [orange]https://github.com/quinn0823/apple-siri-voice-navigation[normal], where you can contribute to it. You can also check for the latest updates there.

[orange]Compatibility[normal]
[blue]1.43+[normal], multiplayer compatible (optional).

[orange]Usage[normal]
1. Click [orange]Mods[normal] on the title screen to open the mod manager. (You are already open! Close this window when proceeding to step 2.)
2. Double-click [orange]Apple Siri Voice Navigation[normal] to activate this mod.
3. Open [orange]Options[normal].
4. Go to [orange]Audio - Voice Navigation - Language and voice[normal] and then select a voice you like.
